## Run Locally

To run the React App(go to directory where file is saved)

```bash
  npm start
```
Install axios

```bash
  npm i axios
```

Install nodemon(backend live server)

```bash
  npm i nodemon
```

Start the server

```bash
  nodemon fileName.js
```
Install cors (backend frontend integration)
```bash
  npm i cors
```
Install mongoDB
```bash
  npm i mongodb
```
